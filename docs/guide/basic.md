# 基础使用

本节介绍 StarHub 的基本使用方法。

## 登录

1. 打开 StarHub 首页
2. 点击 **使用 GitHub 登录** 按钮
3. 在弹出的 GitHub 页面授权
4. 授权成功后自动跳转到主页

## 界面介绍

StarHub 主界面分为三个区域：

```
┌─────────────────────────────────────────────────────┐
│                    顶部导航栏                         │
├──────────┬─────────────────────┬────────────────────┤
│          │                     │                    │
│  左侧栏   │     仓库列表         │     详情面板        │
│  (标签)   │                     │                    │
│          │                     │                    │
└──────────┴─────────────────────┴────────────────────┘
```

### 顶部导航栏

- **搜索框**：输入关键词搜索仓库
- **同步状态**：显示数据同步进度
- **用户菜单**：设置、重新抓取、退出

### 左侧标签栏

- **全部**：显示所有仓库
- **未分类**：显示未添加标签的仓库
- **各分类标签**：按标签筛选仓库
- **AI 分类按钮**：触发智能分类

### 仓库列表

- 显示符合筛选条件的仓库
- 支持虚拟滚动，流畅展示大量数据
- 点击仓库查看详情

### 详情面板

- 仓库基本信息
- Star/Fork/Issue 数量
- 编程语言和标签
- README 预览

## 同步仓库

### 首次同步

首次登录会自动开始同步所有 Star 仓库。

同步进度显示在右上角：

```
正在同步... 150/500
```

:::tip
GitHub API 有速率限制（每小时 5000 次请求），大量数据同步需要一些时间。
:::

### 手动刷新

如果需要获取最新数据：

1. 点击右上角用户头像
2. 选择 **重新抓取**

## 搜索仓库

在顶部搜索框输入关键词，支持搜索：

- 仓库名称
- 仓库描述
- 所有者名称
- 编程语言

### 搜索技巧

| 搜索方式 | 示例 | 说明 |
|----------|------|------|
| 关键词 | `react` | 搜索包含 react 的仓库 |
| 多关键词 | `react hooks` | 同时包含两个词 |
| 语言筛选 | 点击左侧标签 | 配合搜索过滤 |

## 查看详情

点击任意仓库，右侧显示详情面板：

### 基本信息

- 仓库名称和所有者
- 仓库描述
- Star / Fork / Issue 数量
- 主要编程语言
- 创建和更新时间

### 标签管理

- 查看当前标签
- 点击 **+** 添加标签
- 点击标签上的 **×** 移除

### README 预览

点击 **查看 README** 按钮：

- 完整 Markdown 渲染
- 代码语法高亮
- 图片正常显示
- 表格和列表格式化

## 打开仓库

点击仓库卡片上的 **↗** 按钮，在新标签页打开 GitHub 仓库页面。

## 设置

点击右上角用户头像 → **设置**：

### AI 分类设置

- 选择 AI 服务商
- 配置 API Key
- 设置批次大小
- 开启/关闭读取 README

### 数据管理

- 查看数据统计
- 清空所有数据
- 检查数据库状态

## 主题切换

1. 点击右上角用户头像
2. 选择 **主题**
3. 切换深色/浅色模式

## 语言切换

1. 点击右上角用户头像
2. 选择 **语言**
3. 切换中文/英文
