## [1.0.0] - 2024-12-26

### 🎉 首个正式版本

这是 StarHub 的第一个正式发布版本，包含完整的 GitHub Stars 管理功能。

### ✨ 新功能

#### 核心功能
- **GitHub 登录**：通过 GitHub OAuth 安全登录
- **仓库同步**：自动同步所有 Star 仓库，支持增量更新
- **标签系统**：自定义标签管理，支持 Emoji 和颜色
- **全文搜索**：毫秒级本地搜索，支持多维度过滤
- **README 预览**：应用内 Markdown 渲染，支持语法高亮

#### AI 智能分类
- 支持 5 种 AI 服务商（OpenAI、Claude、DeepSeek、通义千问、智谱）
- 可配置批次大小
- 支持读取 README 进行深度分类
- 提供仅未分类/全部重新分类两种模式

#### 用户界面
- 精美的响应式设计
- 深色/浅色主题切换
- 中英文双语支持
- 虚拟滚动支持大量数据
- 炫酷的登录页星空动画

#### 数据管理
- 基于 IndexedDB 的本地存储
- 支持数据清空和重新同步
- 多种数据修复脚本

### 🛠️ 技术实现

- Vue 3 + TypeScript + Vite 技术栈
- Pinia 状态管理
- Element Plus 组件库
- Dexie.js 数据库封装
- Marked + highlight.js Markdown 渲染

### 📝 文档

- 完整的使用文档
- 详细的部署指南
- 故障排除说明
- 代码贡献指南

## 提交规范

本项目使用以下提交类型：

| 类型 | 说明 |
|------|------|
| `feat` | 新功能 |
| `fix` | 修复 Bug |
| `docs` | 文档更新 |
| `style` | 代码风格调整 |
| `refactor` | 代码重构 |
| `perf` | 性能优化 |
| `test` | 测试相关 |
| `chore` | 构建/工具相关 |
